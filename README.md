# Alqartha dia Bot

بوت واتساب متطور مبني على Node.js ومكتبة Baileys.

## الميزات
- **الردود التلقائية**: ردود ذكية على التحية والرسائل الشائعة.
- **إدارة المجموعات**: طرد، ترقية، وتنزيل الرتب للمشرفين.
- **الترحيب والوداع**: رسائل تلقائية عند انضمام أو مغادرة الأعضاء.
- **ترفيه**: ألعاب حجر-ورقة-مقص.
- **أدوات**: تحويل الصور إلى ملصقات (`.sticker`) وتحويل النص إلى كلام (`.tts`).

## التشغيل
1. اضغط على زر **Run** في Replit.
2. امسح رمز **QR Code** الظاهر في الـ Console باستخدام واتساب هاتفك.
